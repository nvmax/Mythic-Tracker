Server Configuration

Enter your Discord server ID and the channel ID where you want to receive Mythic+ run notifications.

Important: Setting a channel is required. The bot will not work in your server until you configure a channel.

To get your server ID, enable Developer Mode in Discord (Settings > Advanced), then right-click on your server icon and select "Copy ID"
To get a channel ID, enable Developer Mode in Discord (Settings > Advanced), then right-click on a channel and select "Copy ID"

Alternative Method

You can also set the notification channel directly in Discord using the /set_channel command:

/set_channel #your-channel

This command must be used by a server administrator.

Mythic+ Tracker Bot Help

Learn how to use the Mythic+ Tracker Bot

Commands

Command Description Example
/track Track a player's Mythic+ runs /track stonehenge deathwing us
/untrack Stop tracking a player /untrack stonehenge deathwing us
/list List all tracked players in your server /list
/check_runs Force check for new runs for a specific character /check_runs stonehenge deathwing us
/check_all Force check for new runs for all tracked characters (admin only) /check_all
/set_channel Set the channel for notifications (admin only) /set_channel #mythic-runs
/ping Check if the bot is responding /ping

Frequently Asked Questions

How often does the bot check for new runs?
The bot checks for new runs every minute by default.
Can I track players from different regions?
Yes, you can track players from any region (US, EU, KR, TW, CN). Just specify the region when using the /track command.
How do I change the notification channel?
Server administrators can use the /set_channel command to change the notification channel, or use the web interface to configure it.
Why am I not seeing notifications?
Make sure:
  • The bot has permission to send messages in the configured channel
  • You've tracked at least one character using the /track command
  • The tracked character has completed a new Mythic+ run since you started tracking them
Can I track multiple characters?
Yes, you can track as many characters as you want. Use the /track command for each character you want to track.

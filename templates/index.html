Mythic+ Tracker Bot

Track your World of Warcraft Mythic+ runs with our Discord bot!

Get Started

Follow these simple steps to add the Mythic+ Tracker Bot to your Discord server:

  1. Click the "Add to Discord" button below to invite the bot to your server
  2. Required: After adding the bot, use the "Configure Bot" button to set up the notification channel
  3. Start tracking your characters with the /track command in Discord

Note: The bot will not work until you configure a notification channel for your server. This is a required step.

Features

  • Track players' Mythic+ runs using the Raider.io API
  • Persistent tracking even if the bot restarts
  • Formatted notifications for completed runs
  • Slash commands for easy interaction
  • Channel restriction to keep bot activity in a designated channel
  • Multi-server support - track different players on different servers

Commands

  • /track <name> <realm> [region] - Track a player's Mythic+ runs
  • /untrack <name> <realm> [region] - Stop tracking a player
  • /list - List all tracked players in your server
  • /check_runs <name> <realm> [region] - Force check for new runs
  • /set_channel #channel - Set the channel for notifications
